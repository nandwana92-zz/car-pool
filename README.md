# Car Pool
App to share your ride with people on your work route.

### Installation

To run the server
```sh
$ cd server
$ npm install
$ node index.js
```

To run the client
```sh
$ cd client
$ npm install
$ gulp serve
```

After the above steps, open this [URL](http://localhost:3000/) in your browser
